# Vitória Régia v4.3.1 — Organização de Configurações, Cadastros e Solicitações

Esta versão reorganiza o sistema para deixar as áreas mais intuitivas:

- Configurações separadas de cadastros.
- Subabas de Configurações: Geral, Canais, Emergência, Apps e Banco/Backup.
- Links clicáveis para Telegram, WhatsApp e e-mail.
- Cadastro de funcionários sem unidade residencial.
- Setores: Administração, Portaria, Zeladoria e Limpeza.
- Nova área de Solicitações para reclamações de limpeza, vazamento, queda de fase de energia e manutenção.
- Fluxo: morador solicita → síndico/administração aprova → equipe de limpeza/zeladoria recebe → conclui.
- Aba Emergências mais visual, com tipos de ocorrência claros.

Sistema desenvolvido em parceria por Bruno Saraiva e ChatGPT.
