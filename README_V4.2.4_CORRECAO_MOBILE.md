# Vitória Régia v4.2.4 — Correção mobile e notificações

Esta versão corrige problemas relatados no celular:

- menu lateral agora rola corretamente;
- itens do menu voltam a aceitar clique/toque;
- menu mantém o visual premium escuro, igual ao restante do sistema;
- overlay do menu não bloqueia os links;
- botão de fechar menu adicionado no mobile;
- central de notificações ganhou botão de fechar;
- notificações podem ser marcadas como lidas e removidas;
- painel de notificações fecha ao marcar como lidas, ao tocar fora ou ao pressionar ESC.

Arquivos novos:

- `vr-mobile-stability.css`
- `vr-mobile-stability.js`

Mantenha estes arquivos carregados por último no `index.html`.
