Use a câmera do celular para ler diferentes modelos de etiqueta, inclusive Jadlog, Correios, transportadoras privadas, QR Code, código de barras e texto impresso.

Em celulares compatíveis, fotografe a etiqueta inteira, de frente, com boa luz e sem reflexo no plástico. O sistema usa leitura reforçada de QR/código de barras, leitura automática em várias regiões da foto e histórico de destinatários para preencher apartamento, destinatário, transportadora e código.

A foto fica no Supabase Storage; o banco salva somente nome, tamanho, tipo e caminho do arquivo.

Integrações

E-mail e WhatsApp automáticos

Preencha as credenciais apenas no servidor. As senhas não aparecem novamente depois de salvas.

E-mail automático

SMTP / Gmail

MailerSend API

Use um remetente validado no MailerSend. O token deve ficar apenas no Render ou salvo pelo painel do síndico.

WhatsApp automático

Evolution API

Use a instância conectada na Evolution API. A API Key deve ficar apenas no Render ou salva pelo painel do síndico.

Periskope API

Informe o número conectado no Periskope no formato com DDI e DDD, sem espaços. A API Key deve ficar apenas no Render ou salva pelo painel do síndico.

Meta WhatsApp Cloud API

Asaas — boletos bancários

O boleto será registrado no Asaas usando CPF/CNPJ do morador. A API Key deve ficar apenas no Render, nunca no GitHub.

Armazenamento externo

Use para guardar fotos e documentos fora do banco de dados. O banco salvará apenas metadados e caminho externo.

Supabase Storage

Use bucket privado com Service Role Key apenas no Render. O frontend nunca recebe a chave.

TeraBox

Use um access_token oficial da TeraBox Open Platform. Não salve esse token no GitHub.
