# Vitória Régia Pro v10.2 — Layout premium com prédio real

Esta versão redesenha o sistema usando a imagem real do Condomínio Vitória Régia, com menu lateral expansível/recolhível, visual premium em desktop e adaptação automática para celular.

## Principais ajustes

- Login com foto real do prédio como fundo.
- Logo redesenhada com inspiração arquitetônica do prédio.
- Menu lateral escuro, expansível e recolhível.
- Dashboard com cards premium, atalhos, permissões e aplicativos.
- Mobile com menu hamburger, navegação inferior e botão de emergência flutuante.
- Mantém funcionalidades da v10.1: notificações, Telegram, atualizações com progresso, central de apps, manuais, reservas, encomendas, cadastros e auditoria.

## Publicação

Use `publicar_github_mac_linux.sh` no Mac ou envie o pacote GitHub Web para a raiz do repositório.
