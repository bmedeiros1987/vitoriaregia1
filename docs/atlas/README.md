# Projeto Atlas — Vitória Régia

O Projeto Atlas organiza a evolução do Vitória Régia como produto da Bruno Medeiros Tecnologia.

## Diretriz central

> O sistema pode oferecer muitos recursos, mas o morador, o síndico e a administração nunca devem se perder.

## Princípios de experiência

- Uma tela deve ter um objetivo principal.
- A informação mais importante deve aparecer primeiro.
- Menus devem permanecer fixos, previsíveis e sem sobrepor o conteúdo.
- Recursos administrativos avançados não devem poluir a experiência cotidiana do morador.
- Resumo primeiro; detalhes sob demanda.
- A mesma linguagem visual e de navegação deve ser mantida em todas as páginas.
- Ações críticas devem ter confirmação, retorno claro e histórico quando aplicável.

## Frentes Atlas

1. Auditoria de arquitetura, rotas, menus e permissões.
2. Design System compartilhado da Bruno Medeiros Tecnologia.
3. Padronização de cabeçalhos, menus internos, cards, formulários e estados vazios.
4. Documentação institucional, segurança, privacidade e transparência.
5. Centro de ajuda e manuais por perfil de usuário.
6. Roadmap, changelog e release notes.
7. Preparação comercial e institucional para condomínios e administradoras.

## Método BMT

Observar → Entender → Simplificar → Construir → Evoluir continuamente.
