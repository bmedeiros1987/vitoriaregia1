# Vitória Régia — Pacote Compacto v4.2.1

Este pacote foi preparado para upload pelo navegador do GitHub com menos de 100 arquivos.

## O que foi compactado

- Projetos Android e iOS foram mantidos como arquivos ZIP dentro da pasta `downloads/`.
- Documentos antigos duplicados foram removidos.
- Foram mantidos apenas os manuais ilustrativos mais recentes.
- O sistema web, backend, MySQL, vídeos da Ajuda, botão de emergência e layout premium foram preservados.

## Como subir

1. Extraia este ZIP.
2. Envie todo o conteúdo extraído para a raiz do GitHub.
3. Não envie a pasta do ZIP por dentro do repositório.
4. Depois faça Manual Deploy no Render.

## Arquivos críticos

Verifique se existem:

- `index.html`
- `app.js`
- `styles.css`
- `backend/src/server.js`
- `backend/src/db.js`
- `render.yaml`
- `VERSION.json`
