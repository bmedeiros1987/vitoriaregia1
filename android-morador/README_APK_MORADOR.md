# APK Vitória Régia Morador

Projeto Android WebView para criar APK do morador.

Abre o sistema em https://vitoriaregia1.onrender.com/#dashboard e mantém login por e-mail/senha.

Funcionalidades principais:

- Encomendas da unidade;
- Notificações/comunicados;
- Últimos visitantes cadastrados no sistema;
- Reservas, serviços e financeiro público;
- Sem armazenar senhas, tokens ou banco dentro do APK.
