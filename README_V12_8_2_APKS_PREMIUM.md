# Vitória Régia Pro v12.8.2 — Central Premium de APKs

Esta versão adiciona uma área premium em Sistema e Apps > Aplicativos para disponibilizar os APKs Android do condomínio.

## O que foi adicionado

- Cards premium para APK da Portaria, Síndico e Morador.
- Logo do Vitória Régia em cada aplicativo.
- Status do APK publicado no sistema.
- Botão de download autenticado.
- Upload de APK por perfis autorizados.
- Instruções internas para publicação na Google Play.
- Limite padrão de upload de documentos ampliado para 150 MB, permitindo APKs maiores.

## Como usar no sistema

1. Entre como usuário master/admin/síndico com permissão de documentos.
2. Acesse Sistema e Apps > Aplicativos.
3. Em cada card, clique em Enviar APK.
4. Envie o arquivo:
   - vitoria-regia-portaria.apk
   - vitoria-regia-sindico.apk
   - vitoria-regia-morador.apk
5. Após o envio, o botão Baixar APK aparecerá para os usuários autorizados.

## Publicação na Google Play

Para Google Play, o recomendado é gerar arquivo Android App Bundle (.aab) assinado. O APK continua útil para testes internos e instalação direta.
