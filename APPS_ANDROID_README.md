# Apps Android Vitória Régia Pro v9.3

O pacote inclui três wrappers Android/WebView:

- `android-portaria`: abre `/?app=portaria#/portaria`
- `android-sindico`: abre `/?app=sindico#/dashboard`
- `android-morador`: abre `/?app=morador#/morador`

## Gerar APKs pelo GitHub

1. Suba o pacote para o repositório.
2. Abra GitHub -> Actions.
3. Execute o workflow **Gerar APKs Android Vitória Régia Pro**.
4. Baixe os arquivos em **Artifacts**.

Os apps solicitam câmera para OCR e foto de visitantes/encomendas.
