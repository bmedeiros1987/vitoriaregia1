# Vitória Régia v4.3.2 — Dashboard por perfil e privacidade

Esta versão melhora a página inicial após o login.

## Principais pontos

- Síndico/administração vê notificações e pendências administrativas logo no dashboard.
- Morador vê somente informações da própria unidade.
- Portaria vê tarefas operacionais.
- Financeiro geral do prédio fica oculto para moradores, salvo quando o síndico publicar algo como público.
- Botões rápidos continuam disponíveis, porém de forma mais limpa e segmentada por perfil.
