# Vitória Régia Pro v12.6.3 — Telegram Portaria Premium

Atualização cumulativa com configuração premium para o Telegram da portaria e fluxo de encomendas.

## Incluído

- Chat ID específico para o celular Telegram da portaria.
- Configuração em **Configurações > Notificações > Telegram Portaria Premium**.
- Telegram da portaria recebe emergências, encomendas, decisões de moradores e avisos operacionais.
- Encomendas enviam botões ao morador: enviar pelo elevador, retirar agora, retirar mais tarde, chamar no interfone ou não reconhecer.
- Quando o interfone não atender, a portaria pode acionar o Telegram do morador pelo sistema.
- Botão “Interfone sem contato” em encomendas e visitantes.
- Configurações para APK/kiosk premium da portaria, com apps liberados e PIN.
- Mensagens seguem o padrão premium usado no módulo de emergência.

## Chat padrão sugerido

- Condomínio/global: `8188648317`
- Portaria: configurável em Notificações, com fallback para o chat global.
