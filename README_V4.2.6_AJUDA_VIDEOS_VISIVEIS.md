# Vitória Régia v4.2.6 — Ajuda e vídeos visíveis

Correção focada na aba **Ajuda**.

## Ajustes

- A aba Ajuda fica visível para todos os perfis autorizados, inclusive administrador/proprietário.
- A seção **Vídeos rápidos** é recriada automaticamente caso algum script anterior esconda ou remova o bloco.
- Os vídeos aparecem no topo da Ajuda, com botão Play e link “Abrir vídeo”.
- Mantidos Telegram, MySQL, manuais, layout premium e correções mobile.

## Onde ficam os vídeos

`Ajuda → Vídeos rápidos`

Arquivos físicos:

`assets/help-videos/*.mp4`
