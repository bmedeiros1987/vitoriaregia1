# Instalação no celular da portaria

## Checklist do aparelho

- Android com câmera funcional.
- Wi-Fi estável.
- Carregador fixo.
- Tela sempre ligada, se possível.
- Telegram instalado com `@portariavr1`.
- APK Vitória Régia Portaria instalado.

## Instalação rápida

1. Gere o APK pelo GitHub Actions ou Android Studio.
2. Transfira o APK para o celular.
3. Toque no APK e permita instalar de fonte desconhecida.
4. Abra o app.
5. Permita câmera.
6. Faça login no sistema com usuário da portaria.
7. Ative fixação de app no Android.
8. Fixe o app Vitória Régia Portaria.

## Para abrir Telegram ou Wi-Fi

1. Toque no botão `⋮` no canto superior direito.
2. Digite o PIN.
3. Escolha:
   - Abrir Telegram da portaria;
   - Abrir Wi-Fi;
   - Abrir câmera;
   - Pausar kiosk/trocar app.
