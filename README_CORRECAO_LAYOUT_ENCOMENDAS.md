# Correção de layout de encomendas

Esta versão corrige o layout da tela **Encomendas**, que podia ultrapassar a largura da página em telas menores, notebooks e WebView do APK.

Ajustes principais:

- a área de encomendas não estoura mais horizontalmente;
- formulário e lista passam para uma coluna em telas médias/pequenas;
- botões da lista de encomendas quebram linha corretamente;
- códigos de rastreio longos passam a quebrar linha;
- cards, status e textos longos ficam limitados à largura da página;
- melhoria de responsividade para celular e APK Android.
