# Vitória Régia - Premium Clean v3.9.0

Esta versão consolida a base MySQL, os módulos restaurados e o layout premium responsivo.

## Principais melhorias

- Menu lateral premium, recolhível e adaptado para PC e celular.
- Tela de login mais limpa, com mínimo de informação.
- Perfil detectado automaticamente pelo usuário e senha.
- Dashboard mais clean, com atalhos clicáveis e visual premium.
- Botão flutuante único de emergência com símbolo de giroflex.
- Notificações internas no próprio app, além de integrações por e-mail, WhatsApp e Telegram.
- Configurações reorganizadas em subabas: Geral, Notificações, Apps e Acessibilidade.
- Opções de acessibilidade para aumentar texto, aumentar contraste e reduzir movimentos.
- Aba Ajuda com manuais simples para moradores, portaria e síndico.
- Seção para baixar aplicativos dentro do sistema.

## Upload pelo PC

Extraia o ZIP e envie todo o conteúdo extraído para a raiz do repositório GitHub.

Confirme que estes arquivos existem na raiz:

```txt
index.html
app.js
styles.css
vr-premium-clean.css
vr-premium-clean.js
backend/src/server.js
backend/src/db.js
VERSION.json
```

Depois faça no Render:

```txt
Manual Deploy -> Deploy latest commit
```

## Banco

A base principal permanece MySQL.

## Observação

Não envie a pasta compactada inteira para dentro do GitHub. Envie os arquivos extraídos diretamente na raiz do projeto.
