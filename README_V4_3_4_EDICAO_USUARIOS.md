# Vitória Régia v4.3.4 — Edição de usuários e senha temporária

Esta versão adiciona:

- Tela de edição de usuário semelhante ao cadastro.
- Alteração de nome, e-mail, telefone, Telegram, unidade, perfil e status.
- Opção de usuário sem unidade vinculada preservada.
- Reset de senha para síndico, administração e administrador proprietário.
- Visualização da senha temporária gerada para copiar.
- Envio da senha temporária pelo Telegram quando o usuário possui Telegram/Chat ID.
- Mensagem de confirmação após salvar.
- Retorno automático para a tela de usuários cadastrados.

Sistema desenvolvido em parceria por Bruno Saraiva e ChatGPT.
