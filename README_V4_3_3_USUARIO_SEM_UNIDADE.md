# Vitória Régia v4.3.3 — Usuário sem unidade vinculada

Esta versão adiciona a opção **“Este usuário não possui unidade vinculada”** nos formulários de cadastro de usuário.

Use para:

- Administração
- Síndico terceirizado
- Portaria
- Zeladoria
- Limpeza
- Funcionários ou prestadores sem apartamento/unidade residencial

Quando marcada, a unidade fica vazia e o sistema entende que o usuário não pertence a uma unidade específica.

Sistema desenvolvido em parceria por Bruno Saraiva e ChatGPT.
