# Vitória Régia v4.2.0 — Ajuda com vídeos por perfil

Esta versão adiciona vídeos explicativos dentro da aba **Ajuda**, separados por perfil de usuário.

## Conteúdo incluído

- Vídeos do Morador
- Vídeos da Portaria
- Vídeos do Síndico/Administração
- Vídeos gerais de Emergência
- Vídeo de Apps e Ajuda
- Manuais ilustrativos v4.1.1
- Seção para baixar aplicativos

## Como subir

Extraia o ZIP e envie todo o conteúdo extraído para a raiz do GitHub. Depois faça o deploy no Render.
