Configuração Render - Vitória Régia Pro v9.3

Root Directory: deixe vazio
Build Command: npm install --no-audit --no-fund && npm run build
Start Command: npm start

Environment Variables obrigatórias:
NODE_VERSION=20.19.0
NODE_ENV=production
DATABASE_URL=sua_url_postgresql
DATABASE_SSL=auto
DATABASE_SSL_MODE=auto
JWT_SECRET=uma_senha_grande_e_forte
APP_VERSION=Vitória Régia Pro v9.3
ADMIN_EMAIL=seu_email_admin
ADMIN_PASSWORD=sua_senha_admin
NPM_CONFIG_AUDIT=false
NPM_CONFIG_FUND=false
NPM_CONFIG_UPDATE_NOTIFIER=false

Push navegador opcional:
VAPID_PUBLIC_KEY=chave_publica_vapid
VAPID_PRIVATE_KEY=chave_privada_vapid
VAPID_SUBJECT=mailto:seu_email

E-mail / SendGrid:
MAIL_PROVIDER=sendgrid
SENDGRID_API_KEY=sua_nova_chave_sendgrid
SENDGRID_FROM_EMAIL=remetente_verificado
SENDGRID_FROM_NAME=Condomínio Vitória Régia
SENDGRID_REPLY_TO=email_de_resposta
SENDGRID_TO_DEFAULT=email_padrao
SENDGRID_DATA_RESIDENCY=global

Telegram:
TELEGRAM_BOT_TOKEN=token_do_bot
TELEGRAM_CHAT_ID=chat_id_padrao_opcional

WhatsApp Cloud API:
WHATSAPP_PHONE_NUMBER_ID=phone_number_id
WHATSAPP_ACCESS_TOKEN=token_cloud_api
WHATSAPP_API_VERSION=v19.0

Depois de alterar variáveis: Manual Deploy -> Clear build cache & deploy.
